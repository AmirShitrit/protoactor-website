Jobb

Jobba med Proto.Actor

Erfaren Go eller .NET Backendutvecklare sökes

Asynkron AB utvecklar opensourceprojektet https://proto.actor, ett ramverk för distribuerade och högtransaktionella system.

Vi arbetar primärt med att hjälpa kunder att implementera system baserat på ramverket. Kundprojekt kan variera från IoT-lösningar med hantering av sensordata, till backends för mobilspel. T.ex. två av världens top 50 största mobilspeltillverkare bygger system med Proto.Actor.

I rollen skulle du bidra med utveckling och underhåll av ramverket, samt arbeta direkt mot slutkund. Uppdelningen är tänkt 50-50 men kan variera över tid.

Krav:

  • Djupa kunskaper inom Go eller .NET - dvs. det är inget krav på kunskap inom både Go och C# från start.
  • Rollen kräver god förståelse för distribuerade system, microservices, event och meddelandebaserad arkitektur.
  • Kunskaper kring prestandatestning, t.ex. Benchmark.NET eller Go test benchmark
  • Grundläggande kunskaper kring Docker och Kubernetes

Meriterande:

  • Erfarenhet av liknande produkter; t.ex. Microsoft Orleans, Dapr, Service Fabric, eller Akka
  • Erfarenhet kring observability; metrics, distribuerad tracing, logging. t.ex. Grafana Loki och Tempo, Jaeger, Zipkin etc.

Språk: Svenska och Engelska

Plats: Remote

Form: Anställning, heltid
